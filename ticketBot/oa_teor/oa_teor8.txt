36. Массив в 1С -это простейшая универсальная коллекция значений, упорядоченных по 
индексу (нумерация начинается с 0). Доступ к элементам массива
осуществляется через индекс, по которому можно получить значение 
элемента.
Пример использования массива в 1С: формирование нового массива,
добавление элементов и их обработка.

// Создание нового массива
Новый Массив();

// Добавление элементов
Массив.Добавить("Строка");
Массив.Добавить(42);
Массив.Добавить('20190315123501');

// Обработка элементов
Сообщить(Массив.Получить(0)); // Вывод первого элемента
Сообщить(Массив.Найти("Строка")); // Поиск элемента "Строка"
Массив.Очистить(); // Очистка массива
