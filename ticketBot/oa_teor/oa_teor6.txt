34. Встроенный язык программирования 1С предоставляет различные операторы для выполнения различных задач. Вот некоторые из них:

1) **Операторы присваивания**: Используются для присваивания значений переменным. Пример: Переменная = 10;

2) **Арифметические операторы**: Используются для выполнения арифметических операций. Примеры: +, -, *, /, % (сложение, вычитание, умножение, деление, остаток от деления).

3) **Логические операторы**: Используются для выполнения логических операций. Примеры: И, ИЛИ, НЕ (логическое И, логическое ИЛИ, логическое НЕ).

4) **Условный оператор ЕСЛИ**: Используется для выполнения блока кода при выполнении определенного условия.
 Пример:

   Если Выражение Тогда
       // Блок кода выполняется, если Выражение истинно
   Иначе
       // Блок кода выполняется, если Выражение ложно
   КонецЕсли
