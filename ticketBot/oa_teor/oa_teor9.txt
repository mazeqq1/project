37.Процедура — это набор команд, который выполняет определённую задачу. Она не 
возвращает значение и используется для выполнения действий.
Функция — это подпрограмма, которая принимает входные данные и возвращает 
выходное значение. Она может выполнять сложные вычисления или 
операции над данными.

Отличия процедур от функций заключаются в следующем:
•Процедура не возвращает значение, в то время как функция возвращает 
результат вычислений.
•В процедуре можно использовать только локальные переменные, в то время 
как функция может использовать как локальные, так и глобальные переменные
•В процедуре можно использовать только один оператор возврата, в то время 
как функция может содержать несколько операторов возврата.
Пример использования процедур и функций в 1С:

Процедура:

Процедура ПечатьСообщения(Сообщение)
 Сообщить(Сообщение);
КонецПроцедуры

Функция:

Функция Сумма(Числа)
 Результат = 0;
 Для каждого Число Из Числа Цикл
 Результат = Результат + Число;
 КонецЦикла;
 Возврат Результат;
КонецФункции
